# xAI Roadmap

## Goal

The goal is to move through the following path:

```text
xAI AI Tutor
  -> AI data quality experience
  -> Deep Learning and Cloud Native portfolio
  -> xAI Model team role
```

## Phase 0. Application Package

### Objective

Prepare an application package for xAI AI Tutor - Korean / Multilingual Audio.

### Deliverables

- English resume
- Cover letter
- Preply evidence summary
- Korean voice samples
- annotated transcript samples
- audio quality rubric
- Korean Voice Data Evaluation Portfolio

### Completion Criteria

- Resume ready
- Cover letter ready
- portfolio PDF ready
- at least 3 voice samples prepared
- application submitted

## Phase 1. Korean Voice Data Evaluation Portfolio

### Objective

Prove the ability to evaluate Korean speech data for AI training.

### Contents

- Standard Korean voice sample
- Natural Korean conversation sample
- English explanation of Korean pronunciation
- annotated transcript
- intonation / rhythm / pronunciation notes
- ambiguous audio judgment cases
- audio quality scoring rubric

### Key Message

I can evaluate not only whether Korean is correct, but whether Korean speech data is useful for AI training.

## Phase 2. Cloud Native AI Evaluation Platform

### Objective

Build the flagship technical portfolio project.

### MVP Features

- create evaluation task
- register text or audio sample metadata
- submit annotation
- submit evaluation score
- mark review_required
- store results in PostgreSQL
- expose FastAPI endpoints
- run with Docker Compose
- deploy to Kubernetes
- test with GitHub Actions

### Technical Stack

- Python
- FastAPI
- PostgreSQL
- Docker
- Docker Compose
- Kubernetes
- GitHub Actions
- Kustomize

## Phase 3. Deep Learning and Evaluation Study

### Objective

Connect human feedback and data quality work to model evaluation.

### Learning Topics

- PyTorch basics
- Transformer basics
- embeddings and attention
- model output evaluation
- RLHF concept
- DPO concept
- reward model concept
- audio ML basics

## Phase 4. Real Service Extension

### Objective

Turn the portfolio project into a real Korean AI evaluation service.

### Service Direction

The service helps users evaluate Korean AI outputs and Korean speech/text data through structured rubrics, human feedback, review workflows, and exportable datasets.

### Initial Service MVP

- register Korean text or speech sample
- evaluate naturalness, accuracy, context fit, and quality
- save evaluation results
- export CSV / JSON
- use sample/demo data first

## Phase 5. xAI Model Team Preparation

### Target Roles

1. Member of Technical Staff - RL Infrastructure
2. Member of Technical Staff - Voice Model
3. Member of Technical Staff - Post-Training and RL

### Final Portfolio Message

I started from Korean tutoring and audio data evaluation, then built a cloud-native evaluation platform to understand annotation workflows, human feedback, data quality, and evaluation infrastructure.
