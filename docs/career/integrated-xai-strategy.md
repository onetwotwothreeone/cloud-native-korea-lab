# Integrated xAI Strategy

## Purpose

This document integrates the xAI career roadmap into Cloud Native Korea Lab.

Cloud Native Korea Lab is no longer only a cloud-native learning repository. It is now a long-term portfolio project for building a cloud-native AI data evaluation platform and preparing for xAI AI Tutor and Model team roles.

## Core Direction

```text
AI Tutor preparation
  -> Korean voice and text data evaluation portfolio
  -> Cloud Native AI Evaluation Platform
  -> real service
  -> xAI Model team application
```

## One-line Positioning

Cloud Native Korea Lab is a cloud-native learning and portfolio project for building AI data evaluation, annotation, feedback, and model evaluation workflows.

## Career Positioning

The target personal positioning is:

```text
Korean AI Data & Evaluation Builder
with Cloud Native Infrastructure Skills
```

This connects the following experiences:

- Korean tutoring experience
- English-based explanation ability
- FSE-style observation, documentation, and troubleshooting
- Cloud native hands-on labs
- AI data evaluation and annotation workflows
- xAI AI Tutor preparation
- Long-term xAI Model team preparation

## Target xAI Roles

Priority order:

1. AI Tutor - Korean / Multilingual Audio
2. Member of Technical Staff - RL Infrastructure
3. Member of Technical Staff - Voice Model
4. Member of Technical Staff - Post-Training and RL

## Why This Integration Matters

Running Cloud Native Korea Lab and the xAI roadmap separately creates scattered effort.

The integrated strategy keeps every activity connected to one question:

> Does this help build evidence for AI data evaluation, cloud-native infrastructure, or xAI readiness?

If yes, it belongs in the project.
If no, it should be postponed.

## Project Identity

```text
mini-platform = practice sandbox
ai-evaluation-platform = flagship portfolio
real service = long-term product
```

The existing FastAPI mini-platform remains useful as a learning sandbox.
The AI Evaluation Platform becomes the main portfolio project.

## Main Deliverables

1. xAI AI Tutor application package
2. Korean Voice Data Evaluation Portfolio
3. Cloud Native AI Evaluation Platform
4. Deep Learning / Evaluation / RLHF learning notes
5. Portfolio-first, service-next product roadmap
6. Public documentation and community content

## Operating Principle

Every lab should produce at least one of the following:

- GitHub commit
- README update
- architecture note
- error log
- portfolio artifact
- Naver Cafe post
- interview story

## Final Goal

The final goal is not only to apply for xAI.

The final goal is to build a real cloud-native AI evaluation service that proves technical ability, product thinking, and operational discipline.
