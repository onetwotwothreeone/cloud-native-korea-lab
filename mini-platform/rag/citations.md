# RAG Citation Plan

## Purpose

This file explains how the AI Docs Agent should show the sources used in an answer.

## Simple Explanation

Citation means showing which official document the answer came from.

## Rules

1. Prefer official documentation.
2. Do not invent sources.
3. If the source is unclear, say so honestly.
4. Keep the explanation easy, but keep the source accurate.

## Answer Format

When possible, answers should include:

```text
공식 문서 기준:
- Source name
- Relevant topic
- Short explanation
```

## Mini Platform Connection

Citations make the AI Docs Agent trustworthy for beginners and useful for portfolio review.
