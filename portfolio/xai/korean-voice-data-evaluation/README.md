# Korean Voice Data Evaluation Portfolio

## Purpose

This portfolio supports the xAI AI Tutor - Korean / Multilingual Audio application.

It demonstrates the ability to evaluate Korean speech data for AI training, focusing on pronunciation, intonation, rhythm, naturalness, ambiguity, transcription quality, and cultural context.

## Positioning

```text
Korean AI Voice Data Evaluator
```

This is not only a Korean tutoring portfolio.
It is an AI training data quality portfolio.

## Portfolio Components

1. Voice samples
2. Annotated transcript samples
3. Audio quality rubric
4. Ambiguous audio judgment log
5. Korean pronunciation and intonation notes
6. Preply tutoring evidence summary
7. Reflection on data quality and model performance

## Audio Quality Rubric

Each sample can be evaluated on a 1-5 scale.

| Category | Meaning |
|---|---|
| Clarity | How easy it is to hear the speech |
| Pronunciation | How accurate and natural the pronunciation is |
| Intonation | Whether pitch movement sounds natural |
| Rhythm | Whether pacing and stress sound natural |
| Noise | Background noise level |
| Transcription Confidence | How confidently the speech can be transcribed |
| Cultural Context | Whether expression fits Korean usage |

## Judgment Log

For ambiguous cases, write:

- sample id
- what was unclear
- possible interpretations
- final decision
- reason
- whether review is required

## Important Rule

Do not include private student information or any confidential data.

Use only self-created voice samples, public samples, or properly anonymized examples.
